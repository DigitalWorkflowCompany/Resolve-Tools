# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2024-12-31

### Added
- CCC-Import: Color correction file import utility
- Create-Bins: Bin creation automation tool
- Create-Dailies: Dailies generation script
- Create-Stills: Still frame extraction utility
- Get-QCNotes: QC notes retrieval tool
- Get-Size: Size information utility
- Import-Shots: Shot import automation
- Rename: Clip renaming utility
- ResolveGrades: Collection of grade preset files (.drx)
  - Dailies base grade
  - ARRI LogC3 dailies preset
  - ARRI LogC4 dailies preset
  - RED Log3G10 dailies preset
  - Sony S-Log3 dailies preset
  - Sony FilmLike dailies preset
- DaVinci Resolve Scripting API documentation
